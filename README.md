## Getting Started

```bash
# Clone this repository
$ git clone https://github.com/nandu-k01/CV-2.git
# Go into the repository
$ cd CV-2
# install mediapipe and opencv2
$ pip install mediapipe
$ pip install opencv-python
# Run the app
$ python min.py
```
